There are 3 types of lists:

Definition lists
Which show terms and definitions
Unordered lists
Which show bullets
Ordered lists
Which show numbers or letters

Above is an example of a definition list.

Unordered list can have a type attribute which can be applied to the UL or the LI tag. Tje three values allowed are:

  • type = "square"
  • type = "disk"
  • type = "circle"

Ordered list can have a type attribute too. Used along with a start attribute they can show:

  1. lower case letters
  2. upper case letters
  3. lower case roman numerals
  4. uppercase roman numerals
  5. numbers
